Responsive list of selectable items

Description:

Responsive list of items - 1-2-3 columns

Input

Properties:
  • config: {
    id: "ishav"
    label: "Ishav"
    thumb: "36c324ecb80dfd40c6c1.jpg"
    }
Attributes:
  • align-image: 'left' | 'right'
  • Aligns thumbnail left or right in the item

Events

item-click: returns selected item-id

--css-variables

  • --item-color: #70926f
  • --item-color-dark: #485f48bd;
  • --item-color-light: rgba(76, 175, 80, 1);
  • --overlay: #81da9130;
  • --background-gradient: linear-gradient( 94deg,rgba(76, 175, 80, 1) 0%,rgba(212, 212, 0, 0.92) 49%, rgba(76, 175, 80, 1) 100%);
  • --title-color: darkslategray;
  • --subtitle-color: darkslategray;

Example

<ph-list align-image="left"></ph-list>

Radio Button Group

Description:

Creates radio-button group from array

Input

Properties:
  • config: {
    • groupHeading: string,
    • fieldName: string, - name given to all radiobuttons in group/li>
    • selectedId: any (id)
    • radiobuttons: {
      • label: string;
      • id: any;
      • value: any
      }[]
Attributes:
  • flex-direction: 'horizontal' | 'vertical'
  • Aligns radio-buttons in group in row or column

Events

change: returns radiobutton value

--css-variables

  • --radio-button-active-color
  • --radio-button-color

Example

<ph-radio-button-group flex-diretion="vertical"></ph-radio-button-group>
